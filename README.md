# 🚀 Laporin Aja Bro

A modern web-based public reporting system that helps citizens report public service issues easily and efficiently.

---

## 📌 Status

🚧 This project is currently under development.

---

## ✨ Features (Planned)

* 📝 Public issue reporting (infrastructure, services, etc.)
* 🤖 AI-powered report classification
* 🕵️ Anonymous reporting option
* 📊 Admin dashboard for report management
* ⚡ Smart insights & summaries for faster decision making

---

## 🛠️ Tech Stack

* Next.js (App Router)
* Tailwind CSS
* Supabase (Database & Auth)
* AI Integration (planned)

---

## 🎯 Goal

To improve public service efficiency by providing a smart, accessible, and user-friendly reporting platform.
